# systeme-information-sothema
Ce projet présente la conception et la mise en place d’un système d’information complet destiné à l’entreprise pharmaceutique SOTHEMA, spécialisée dans la fabrication de produits médicaux.
L’objectif est d’optimiser la gestion des employés, des projets de production, des fournisseurs, des équipements et des commandes grâce à une base de données relationnelle structurée.
## Objectifs du projet
- Centraliser et organiser les données essentielles de l’entreprise.
- Améliorer la traçabilité des opérations (projets, commandes, équipements…)
- Assurer une meilleure coordination entre les différents départements.
- Offrir une base professionnelle pour le suivi, la planification et l’analyse.
## Contenu du projet
1. Dictionnaire de données
Définition détaillée de toutes les tables, champs, types, contraintes et règles de gestion (Quantitatif/qualitatif, Permanent/temporaire, Interne/externe).
Tables incluses :
- Employé
- Projet
- Fournisseur
- Équipement
- Commande
2. MCD & MLD
- MCD (Modèle Conceptuel de Données) : représentation graphique des entités et de leurs relations.
- MLD (Modèle Logique de Données) : transformation du MCD en tables relationnelles formelles.
3. Création des tables (SQL)
  Le projet inclut les scripts de création des tables avec clés primaires, clés étrangères et types appropriés.
4. Requêtes SQL
  Des requêtes permettant :
- d’afficher la liste des employés,
- de suivre les projets et leurs responsables,
- d’analyser les fournisseurs et leurs commandes,
5. Formulaires (Access)
  Formulaires interactifs pour la saisie, la mise à jour et la consultation rapide des données :
  Formulaire Employé
- Formulaire Projet
- Formulaire Fournisseur
- Formulaire Commande
- Formulaire Équipement
6. États / Rapports
  Rapports automatiques générés à partir de la base de données pour faciliter la prise de décision.
## Technologies utilisées
- Microsoft Access (base de données + formulaires + états)
- SQL
- Conception UML / Merise
## Résultat final
Un système d’information opérationnel et documenté permettant à l’entreprise SOTHEMA :
de gérer efficacement son personnel,
- de suivre l’avancement des projets,
- d’organiser ses relations fournisseurs,
- de maîtriser les stocks et équipements,
- d’automatiser l’analyse des commandes.
de contrôler les stocks d’équipements.
