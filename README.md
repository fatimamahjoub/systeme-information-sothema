# systeme-information-sothema
Projet académique portant sur la conception et la mise en place d’un système d’information intégré pour l’entreprise pharmaceutique SOTHEMA. Le projet inclut l’analyse des besoins, le dictionnaire de données, le MCD/MLD, les tables, les requêtes, les formulaires et la gestion des processus internes afin d’améliorer l’organisation et la performance.
